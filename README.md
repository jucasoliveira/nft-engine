# Quick start

Go to poject directory
```bash
cd ./project
```

Install dependencies

```bash
yarn
```
or 
```bash
npm install
```

## Run development

```bash
yarn dev
```
or
```bash
npm run dev
```

Project might appear on your localhost : http://localhost:3000
